# Description
This project demonstrates file handling operations and basic CRUD (Create, Read, Update, Delete) functionality. The Python script included is a Hardware Inventory System that reads data from a local text file, allows the user to add or remove items, and saves the updated inventory back to the file.

# Project Struction
The repository is organized following standard software development practices:
- src: Contains the main Python source code.
- data: Contains the text files used for storing the database
- README.md: Project documentation and instructions.

# Tool Used
- **OneCompiler:** Online Integrated Development Environment (IDE) used for writing and testing the Python script.
- **GitHub:** Used for version control and maintaining a history of meaningful commits.
- **Python 3:** The programming language used for the logic implementation.

# How to Run the Activity
- Clone the repository to your local machine.
- Ensure Python 3 is installed.
- Navigate to the src/ directory in your terminal.
- Run python inventory.py to start the interactive console menu.

# Test Cases

- **Test 1:** Application successfully loads existing data from items.txt.

- **Test 2:** User adds a new item ("Motherboard") and it correctly appends to the list.

- **Test 3:** Deleting a non-existent item triggers the correct error handling message.
